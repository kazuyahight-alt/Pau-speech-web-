# Pau Speech Web

Simple text to speech web.

## Features

- Text input
- Browser speech
- AI MP3 generation
- Voice selector
- Speed control
- Pause, resume, stop
- Download MP3
- Simple UI

## Deploy to Vercel

1. Upload all files to GitHub root.
2. Import repo to Vercel.
3. Open Project Settings.
4. Open Environment Variables.
5. Add:

```env
OPENAI_API_KEY=your_api_key_here
